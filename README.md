# MSWDO Solo Parent Management System

The Solo Parent Information Management System is a specialized digital platform developed for the Municipal Social Welfare and Development Office (MSWDO) in Pagsanjan. Its primary objective is to modernize and streamline the registration and data management processes for solo parents in accordance with Republic Act 11861.

This system includes a Laravel backend API and a React frontend application.

## Project Structure

- **pagsanjan-backend**: Laravel API handling data management and business logic.
- **pagsanjan-frontend**: React Client providing the user interface.
- **tools**: Utility scripts and tools.

## Getting Started

### Prerequisites

- [XAMPP](https://www.apachefriends.org/) (with PHP and MySQL)
- [Node.js](https://nodejs.org/)
- [Composer](https://getcomposer.org/) (for backend dependencies)

### Installation & Setup

1. **Backend Setup:**
   See [BACKEND_SETUP_GUIDE.md](BACKEND_SETUP_GUIDE.md) for detailed backend configuration.

2. **Frontend Setup:**
   Navigate to `pagsanjan-frontend` and run:
   ```bash
   npm install
   ```

### Running the System

See [HOW_TO_RUN.md](HOW_TO_RUN.md) or execute the provided batch scripts:
- Double-click `start-backend.bat` (or `RUN_SYSTEM.bat`) to launch the backend.
- Double-click `start-frontend.bat` to launch the frontend.
